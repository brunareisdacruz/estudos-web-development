# 🚀 Meus Primeiros Passos no HTML

Bem-vindo ao meu repositório de estudos de Front-end! Este espaço foi criado para documentar minha jornada aprendendo HTML do zero, concentrando projetos simples e exercícios fundamentais.

O objetivo aqui é solidificar a base de estruturação semântica antes de avançar para novos desafios.

## 📌 Projetos no Repositório

Atualmente, o repositório contém:

1. **Heading Elements (Sumário de Livro)**:
   - **O que é:** Uma estrutura de capítulos e subcapítulos.
   - **Foco:** Praticar a hierarquia correta de títulos utilizando de `<h1>` até `<h6>`.

2. **Paragraph Elements**:
   - **O que é:** Pequenos blocos de texto.
   - **Foco:** Entender a marcação de parágrafos com a tag `<p>`.

3. **Angela's Cinnamon Roll Recipe**: 
   - **O que é:** Uma página de receita completa.
   - **Foco:** Praticar listas não ordenadas (`<ul>`) para ingredientes e o uso de listas ordenadas (`<ol>`) com itens de lista (`<li>`) para o passo a passo.

4. **Movie Ranking (Top 3 Melhores Filmes)**:
   - **O que é:** Um ranking dos meus filmes favoritos.
   - **Foco:** Utilização conjunta de listas ordenadas e listas desordenadas para organizar as informações.

---

## 🛠️ Tecnologias Estudadas
* **HTML5:** Tags de texto (`h1-h6`, `p`), listas (`ul`, `ol`, `li`) e estrutura básica.
* **Git & GitHub:** Versionamento e organização de código.

## 🎯 Objetivos de Aprendizado
- [x] Entender a estrutura básica de um documento HTML.
- [x] Dominar a hierarquia de títulos e parágrafos.
- [x] Aplicar listas ordenadas e desordenadas corretamente.
- [ ] Aprender a inserir links (`<a>`) e imagens (`<img>`).
- [ ] Iniciar estudos de tabelas e formulários.
